{# SEO template tag usage examples ================================= 1) Global setup in base.html (recommended — single render, no duplicates): {% load seo_tags %} {% render_seo_meta seo_object %} {% render_json_ld seo_object %} 2) Blog article — pass seo_object and og_type in the view: # views.py return render(request, "frontend/blog_detail.html", { "post": post, "seo_object": post, "seo_og_type": "article", }) # blog_detail.html — nothing extra needed; base.html renders all tags. {% extends "base.html" %} 3) Static page — override defaults via seo_overrides: # views.py return render(request, "frontend/kontakt.html", { "seo_overrides": { "title": "Kontakt", "description": "Pozovite nas za ponudu.", }, }) 4) Explicit article object in template: {% render_seo_meta post "article" %} 5) Read SEO values without HTML (debug/custom partials): {% resolve_seo post "article" as seo %}
